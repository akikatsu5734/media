## レイアウトファミリー別 描き方の補足

| レイアウト | 描き方の要点 |
|---|---|
| single_scene | 1つの印象的な場面。主役中心、補助要素2〜3個、余白少なく仕上げる |
| central_storyboard | 中央主題を大きく配置し、周囲に4〜8個の補助要素・小物をバランスよく配置 |
| infographic_scene | メイン場面（画面の約60%）＋2〜3個の補助場面または補助小物群 |
| center_with_corners | 中央主題＋四隅や周辺にカテゴリを表す要素を均等に配置 |
| consultation_card | 机上・相談シーン。人物・家模型・小物・書類などが整然と並ぶ |
| process_collage | 流れ・比較・注意点。複数の小場面（3〜6個）がコラージュ的に配置 |

## 最終出力指示

「{{記事タイトル}}」の画像を、上記カテゴリ別設計指示をすべて守って生成してください。

**構図・密度（最重要）**
- 指定された layout_family に従い、中央主題＋周辺補助要素の構成にすること
- 静かな1シーンだけで終わらせない。density に応じた補助要素・小場面を必ず加える
- 画面の70〜90%を主題関連要素で埋める。右側・上部に大きな白い空白を残さない

**人物・プロップ**
- people_mode に従い、人物または手元を必要に応じて積極的に使う
- allow_blank_documents が true の場合、文字のない書類・クリップボードを使って情報密度を上げる
- 読める文字・数字・ロゴは一切入れない（blank propの形のみOK）

**スタイル・仕上がり**
- 水彩・手描き・やさしい色味で仕上げる
- 縮小表示でもカテゴリ差が分かる、カード型サムネイルとして仕上げる
- rounded card feel / visually rich but organized
- 不安を煽らず、整理できそうな印象にする
